# 通話許可控制 Cmdlet

 

_**上次修改主題的時間：** 2012-03-21_

通話許可控制 (CAC) 會判斷是否有足夠的網路頻寬可建立品質在可接受的即時音訊或視訊工作階段。您可以透過為網路、網站和子網路 (以及它們之間的互動) 設定限制並進行設定來管理 CAC。
