# 進階 Enterprise Voice Cmdlet

 

_**上次修改主題的時間：** 2012-03-21_

在組織中實作 Enterprise Voice 的同時，您也可以實作其他功能與應用程式，以便提升使用者經驗和組織效率。這些功能包括通話許可控制、增強型 9-1-1 (E9-1-1) 緊急通話和媒體組態設定。
