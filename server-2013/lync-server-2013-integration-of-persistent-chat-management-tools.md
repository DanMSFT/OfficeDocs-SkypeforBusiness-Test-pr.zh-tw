# 在 Lync Server 2013 中整合常設聊天室管理工具

 

_**上次修改主題的時間：** 2012-08-16_

Lync Server 2013 整合其系統管理工具與用於其餘 Lync Server 的工具，簡化了 常設聊天室伺服器 的管理。

常設聊天室伺服器 包含與 Lync Server 控制台整合的系統管理使用者介面經驗。此外， 常設聊天室伺服器 也包含 Windows PowerShell Cmdlet 的集合，以管理 常設聊天室伺服器 類別、聊天室 (包括刪除聊天室和清除過時內容) 和增益集。
