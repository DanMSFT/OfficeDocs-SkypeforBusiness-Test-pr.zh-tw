# 在 Lync Server 2013 中檢閱管理員報告

 

_**上次修改主題的時間：** 2013-02-21_

系統管理員報告包含部署和作業的詳細資訊。這些報告是根據在 \[設計網站\] 中所標記的選擇而產生的。設計人員可以編輯網路圖，並且定義伺服器、集區和負載平衡器的完整 IP 位址和完整網域名稱 (FQDN)，以在系統管理員報告中加入更多的值。
