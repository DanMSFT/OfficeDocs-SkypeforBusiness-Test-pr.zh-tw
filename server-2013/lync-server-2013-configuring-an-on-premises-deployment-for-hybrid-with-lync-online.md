# 設定內部部署以混合使用 Lync Online

 

_**上次修改主題的時間：** 2014-05-28_

混合部署是部分位於內部部署之使用者和部分位於線上之使用者的部署，但是所有的使用者皆共用相同的網域，例如 user@contoso.com。本節將引導您完成部署混合部署所需的應用程式，然後將部署設定為加以啟用。
