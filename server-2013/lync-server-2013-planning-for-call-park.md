# 在 Lync Server 2013 中規劃通話駐留

 

_**上次修改主題的時間：** 2012-09-07_

Lync Server通話駐留應用程式 可讓 Enterprise Voice 使用者保留通話，然後稍後再從其他電話擷取。駐留通話的使用者可以撥打「通話駐留」提供的軌道號碼，以擷取駐留的通話，或使用外部機制 (例如立即訊息或傳呼系統) 向其他人詢問要擷取的號碼。本節包括 通話駐留 專用的規劃資訊。
