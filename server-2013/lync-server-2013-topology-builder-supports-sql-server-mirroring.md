# Lync Server 2013 中的拓撲產生器支援 SQL Server 鏡像

 

_**上次修改主題的時間：** 2012-09-26_

Lync Server 2013 支援高可用性的後端伺服器鏡像。您可以使用 拓撲產生器設定鏡像，包括設定見證。
