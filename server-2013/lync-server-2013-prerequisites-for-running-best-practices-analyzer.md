# 執行最佳做法分析程式的先決條件

 

_**上次修改主題的時間：** 2012-06-25_

您可以使用 Best Practices Analyzer 掃描 Lync Server 2013 環境中是否有任何問題並建立報告，以及檢視目前或之前掃描的結果。執行 Best Practices Analyzer 掃描環境之前，您必須確認要用來執行掃描的帳戶具有必要的權限，並符合所有安裝先決條件。
