# 信任的應用程式 Cmdlet

 

_**上次修改主題的時間：** 2012-06-20_

信任的應用程式是由協力廠商所開發的應用程式，其處於受信任狀態而可在Microsoft Lync Server 2013中執行，但不是產品的內建部分。Lync Server 2013 提供各種 Cmdlet，可用來設定和管理信任的應用程式。

## 信任的應用程式 Cmdlet

使用下列 Cmdlet 來管理信任的應用程式。
