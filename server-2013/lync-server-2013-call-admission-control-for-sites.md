# 網站的通話許可控制

 

_**上次修改主題的時間：** 2013-02-21_

網站是指通話許可控制 (CAC)、E9-1-1 和媒體旁路部署的每一個網路地區內的辦公室或位置。請使用本節中的程序來設定網站的通話許可控制。
