# Web 會議 Cmdlet

 

_**上次修改主題的時間：** 2013-07-29_

會議與線上會議是 Microsoft Lync Server 2013 中的重要項目。CsConferencingConfiguration 和 CsConferencingPolicy Cmdlet 是使用 Windows PowerShell 來管理會議的主要系統管理工具。

## Web 會議 Cmdlet

[New-CsClientPolicy](new-csclientpolicy.md) 和 [Set-CsClientPolicy](set-csclientpolicy.md) Cmdlet 提供設定與管理 Lync Server 2013 的額外方式。
