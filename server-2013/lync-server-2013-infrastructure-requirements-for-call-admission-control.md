# Lync Server 2013 中的通話許可控制的基礎結構需求

 

_**上次修改主題的時間：** 2012-08-21_

不需要特殊網路路由器之類的額外基礎結構需求。部署 Lync Server 2013 時，會自動安裝頻寬原則服務以實施通話許可控制 (CAC) 原則。通話許可控制只能和執行 Lync 的用戶端一起運作。
