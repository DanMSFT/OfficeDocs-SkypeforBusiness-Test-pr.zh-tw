# 使用者移轉

 

_**上次修改主題的時間：** 2012-10-19_

一般可接受的移轉最佳做法是，建立幾位測試使用者，並使用這些使用者進行系統測試。成功移動並測試這些帳戶之後，您應指定一組測試生產使用者，然後移動他們的帳戶並進行驗證測試。當您得到滿意的結果後，就可以將其餘使用者移動至新部署。
