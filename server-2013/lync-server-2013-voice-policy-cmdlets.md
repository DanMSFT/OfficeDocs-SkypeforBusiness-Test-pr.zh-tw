# 語音原則 Cmdlet

 

_**上次修改主題的時間：** 2013-02-18_

Enterprise Voice 的管理作業包括設定如語音原則和撥號對應表等項目，以及使語音原則和語音路由相關聯。與語音原則管理作業相關的 Cmdlet 可用來設定如同時響鈴 (每當有人撥打您的辦公室電話時，讓第二支電話響起的功能)、來電轉接及撥號需求等功能。
