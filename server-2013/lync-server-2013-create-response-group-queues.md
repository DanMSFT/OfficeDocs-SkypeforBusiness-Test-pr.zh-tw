# 在 Lync Server 2013 中建立回應群組佇列

 

_**上次修改主題的時間：** 2012-01-18_

佇列會使來電者等候，直到代理接聽電話為止。當 回應群組應用程式搜尋有空的代理時，它會以您列出的順序搜尋代理群組。您可以選取指派給佇列的代理群組，並指定佇列行為，例如限制佇列中可以包含的電話數目，以及限制在代理接聽電話之前，電話等候的時間長度。
