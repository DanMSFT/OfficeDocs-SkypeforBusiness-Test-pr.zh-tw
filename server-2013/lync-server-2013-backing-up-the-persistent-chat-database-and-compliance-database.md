# 在 Lync Server 2013 中備份常設聊天室資料庫及規範資料庫

 

_**上次修改主題的時間：** 2012-08-01_

您應該定期備份 常設聊天室資料庫與規範資料庫。請與您的資料庫管理員連絡，以決定在您的環境中進行備份的最佳方式。
