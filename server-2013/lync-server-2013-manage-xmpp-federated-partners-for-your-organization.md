# 在 Lync Server 2013 中管理 XMPP 同盟夥伴

 

_**上次修改主題的時間：** 2012-10-19_

此為初稿文件，可能隨時變更。空白主題是內容的預留位置。

若要管理 XMPP 同盟網域使用者的支援，您需要執行下列動作：

  - 設定一個或多個外部存取原則，以支援 XMPP 同盟網域的使用者。

  - 設定「Access Edge 設定」原則以支援同盟。

  - 建立 XMPP 同盟協力廠商定義。

  - 了解 XMPP 同盟可使用的交涉設定。

若要執行這些工作，請利用本節中的程序。
