# 憑證和驗證 Cmdlet

 

_**上次修改主題的時間：** 2012-10-04_

憑證和驗證 Cmdlet 涵蓋範圍廣大的工作包括伺服器和用戶端憑證的管理、使用者 PIN (個人識別碼) 的管理，以及搭配 Internet Information Services 使用之 SIP 網域和 Kerberos 帳戶的管理。
