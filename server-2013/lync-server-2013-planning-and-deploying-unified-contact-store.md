# 在 Lync Server 2013 中規劃和部署整合連絡人存放區

 

_**上次修改主題的時間：** 2012-06-14_

整合連絡人存放區功能可跨 Microsoft Office 產品提供一致的連絡人體驗。此功能可讓使用者在 Exchange 2013 儲存所有連絡人資訊，以便可跨 Lync、 Exchange、Outlook 及 Outlook Web Access 全面通用資訊。
