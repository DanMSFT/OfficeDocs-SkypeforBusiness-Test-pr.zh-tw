# 會議 Cmdlet

 

_**上次修改主題的時間：** 2012-10-09_

Microsoft Lync Server 2013 可讓使用者透過兩種不同方式加入會議：使用會議應用程式 (如 Lync 2013)，或使用電話撥入。撥入的使用者無法執行檢視投影片或交換立即訊息等動作，但可以完整參與會議的音訊部分。

## 會議 Cmdlet

**CsDialInConferencing** Cmdlet 可用以設定各種電話撥入式會議內容，包括指定使用者可撥打來加入會議的電話號碼，或使用者加入會議後所能使用的鍵台命令 (例如，按 6 可將其電話靜音取消靜音) 等。其餘大部分的會議功能 (例如，使用者是否可錄下會議、使用者是否可在會議期間共用應用程式等) 均可透過 **CsConferencingPolicy** Cmdlet 來管理。
