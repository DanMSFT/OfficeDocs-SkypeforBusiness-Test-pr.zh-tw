# 封存與監控 Cmdlet

 

_**上次修改主題的時間：** 2012-10-04_

封存和監控 Cmdlet 可讓系統管理員管理立即訊息和會議工作階段封存；記錄通話詳細資訊；以及使用經驗品質 (QoE) 監控 Microsoft Lync Server 2013。
