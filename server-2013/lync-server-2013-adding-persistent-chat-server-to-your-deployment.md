# 在 Lync Server 2013 中將常設聊天室伺服器新增至您的部署

 

_**上次修改主題的時間：** 2012-09-12_

在您計劃部署 Lync Server 2013、 常設聊天室伺服器 的每台伺服器上安裝先決條件軟體之後，必須使用 拓撲產生器將 常設聊天室伺服器 支援新增至您的拓撲，然後再發行拓撲。
