# Lync Server 2013 中的行動性支援

 

_**上次修改主題的時間：** 2012-06-29_

Lync Server 2013 行動性可讓使用者在自己的行動裝置上存取 Lync 2013 功能。支援的行動裝置包括：

  - 支援的 Apple iOS 裝置

  - Windows Phone

  - Android

  - Nokia

如需有關行動功能和需求的詳細資訊，請參閱規劃文件中的 [Lync Server 2013 中的行動規劃](lync-server-2013-planning-for-mobility.md)。
