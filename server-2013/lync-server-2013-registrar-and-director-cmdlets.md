# 登錄器與 Director Cmdlet

 

_**上次修改主題的時間：** 2012-10-09_

登錄器和 Director 可用來驗證登入要求，並維護使用者狀態與可用性的相關資訊。登錄器與 Director 的 Cmdlet 可讓您管理這些伺服器的組態設定。
