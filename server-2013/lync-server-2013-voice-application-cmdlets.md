# 語音應用程式 Cmdlet

 

_**上次修改主題的時間：** 2012-10-09_

語音應用程式會執行於前端伺服器上的應用程式服務。這些應用程式可讓您保留通話、處理未指派的號碼、與 Exchange 整合通訊 (UM) 協調以及執行 回應群組應用程式，協助您管理來電和撥出電話。
