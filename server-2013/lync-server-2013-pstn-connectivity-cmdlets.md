# PSTN 連線 Cmdlet

 

_**上次修改主題的時間：** 2012-10-09_

Enterprise Voice 提供許多設定，可允許與公用交換電話網路 (PSTN) 來回進行通話，而不會降低服務品質 (QoS)。您可以透過 Lync Server 管理命令介面中提供的 Cmdlet 來進行這些設定。
