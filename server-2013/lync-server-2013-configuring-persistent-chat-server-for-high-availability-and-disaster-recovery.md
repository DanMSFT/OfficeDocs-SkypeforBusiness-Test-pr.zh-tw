# 在 Lync Server 2013 中針對高可用性和災害復原設定常設聊天室伺服器

 

_**上次修改主題的時間：** 2012-10-01_

Lync Server 2013、 常設聊天室伺服器 服務使用 *「延伸集區」* (stretched pool) 組態進行災害復原。延伸集區的電腦分散在兩個實體資料中心之間，但都在單一的邏輯 Lync Server 網站內。
