# 在 Lync Server 2013 中設定進階 E9-1-1 功能

 

_**上次修改主題的時間：** 2012-06-06_

Lync Server 2013 包含下列可供您自訂 E9-1-1 部署的功能：

  - 用以將位置資訊伺服器連線至 SNMP 應用程式的 Web 服務介面。

  - 用以連線至「次要位置來源」資料庫的 Web 服務介面。
