# 指派每位使用者的原則

 

_**上次修改主題的時間：** 2012-10-14_

您可以將特定原則指派給某個使用者或一組使用者，以便指定特定的設定，這些設定偏離在指派給其他使用者之原則 (例如全域原則) 中所定義的設定。這些原則稱為個別使用者原則。
