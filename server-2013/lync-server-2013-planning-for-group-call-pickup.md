# 規劃 Lync Server 2013 中的群組來電接聽

 

_**上次修改主題的時間：** 2013-02-01_

Lync Server 2013 累計更新 (2013 年 2 月) 已導入群組來電接聽做為全新企業語音功能。群組來電接聽讓使用者可藉由撥打來電接聽群組號碼，接聽其他使用者響鈴的來電。本節包含群組來電接聽的特定規劃資訊。
