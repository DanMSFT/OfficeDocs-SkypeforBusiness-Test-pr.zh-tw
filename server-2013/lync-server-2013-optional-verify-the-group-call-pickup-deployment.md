# (選用) 確認群組來電接聽部署

 

_**上次修改主題的時間：** 2013-01-30_

為使用者啟用群組來電接聽之後，請確認設定是否如預期般運作。請至少確認下列項目：

  - 呼叫已啟用群組來電接聽的使用者，並讓另一位使用者擷取來電。該位使用者可位於相同群組、不同群組，或未啟用群組來電接聽。

  - 呼叫已啟用群組來電接聽的使用者，並且不接聽來電。
