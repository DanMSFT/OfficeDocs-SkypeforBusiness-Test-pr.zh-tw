# Persistent Chat Server Cmdlet

 

_**上次修改主題的時間：** 2012-06-27_

常設聊天 Cmdlet 讓您能夠管理和設定 Microsoft Lync Server 2013 常設聊天服務 (先前稱為 \[群組聊天\] 服務)。常設聊天讓使用者能夠參與線上立即訊息工作階段。儘管這些工作階段是即時發生的，但每個工作階段的內容都是持續性；這表示任何人隨時都可以繼續進行這些交談。
