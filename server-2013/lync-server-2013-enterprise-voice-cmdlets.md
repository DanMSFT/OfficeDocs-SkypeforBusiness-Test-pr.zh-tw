# Enterprise Voice Cmdlet

 

_**上次修改主題的時間：** 2014-03-19_

Enterprise Voice 是 Microsoft 實作的 VoIP。Microsoft Lync Server 2013 中用來管理 Enterprise Voice 的 Cmdlet 將可讓您建立及修改撥號對應表 (先前稱為位置設定檔)、語音原則、路由和正規化規則。

## Enterprise Voice Cmdlet

Enterprise Voice 的管理工作大多可從 Lync Server 控制台執行。這些工作也全都可使用 Lync Server 管理命令介面 中的 Cmdlet 執行，或在指令碼內執行 (可讓您將特定工作自動化)。以下列出與管理 Enterprise Voice 直接相關的 Cmdlet：
