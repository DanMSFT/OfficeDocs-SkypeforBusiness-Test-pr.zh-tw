# Lync Server 2013 中的用戶端和裝置硬體支援

 

_**上次修改主題的時間：** 2012-10-01_

用戶端電腦必須符合特定硬體需求，以支援 Lync Server 2013 部署中的用戶端。額外硬體設定必須已就緒，才能部署 IP 電話和類比裝置。
