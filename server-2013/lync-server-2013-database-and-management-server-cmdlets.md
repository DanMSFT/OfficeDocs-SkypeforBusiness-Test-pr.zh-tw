# 資料庫和管理伺服器 Cmdlet

 

_**上次修改主題的時間：** 2012-10-09_

資料庫和管理伺服器 Cmdlet 是用來管理 Microsoft Lync Server 2013 後端資料庫和前端管理服務。您可以使用這些 Cmdlet 來安裝或解除安裝 Lync Server 2013 使用的任何資料庫，以及設定中央管理存放區的 Active Directory 服務控制點。

## 資料庫和管理伺服器 Cmdlet
