# 連接埠需求

 

_**上次修改主題的時間：** 2013-03-27_

Lync Server 需要開啟防火牆上的特定連接埠。此外，如果組織中已部署 Internet Protocol Security (IPsec)，則用於傳送音訊、視訊與全景視訊的連接埠範圍都需停用 IPsec。
