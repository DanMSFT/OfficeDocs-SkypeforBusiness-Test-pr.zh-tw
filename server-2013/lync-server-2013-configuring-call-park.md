# 在 Lync Server 2013 中設定通話駐留

 

_**上次修改主題的時間：** 2012-10-30_

通話駐留可讓 企業語音使用者在某一支電話上保留通話，並於稍後從另一支電話撥打內部號碼 ( 通話駐留*「軌道」* ) 接聽該通話。

通話駐留使用的元件會在部署 企業語音時，於 前端伺服器或是 Standard Edition 伺服器 上自動安裝並啟用。不過，您必須先設定 通話駐留，才能對使用者提供。

本節將引導您設定 通話駐留。
