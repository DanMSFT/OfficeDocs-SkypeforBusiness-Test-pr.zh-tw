# Lync Server 2013 中的容錯移轉和容錯回復常設聊天室伺服器

 

_**上次修改主題的時間：** 2012-08-03_

若想對 Lync Server 2013、 常設聊天室伺服器 進行容錯移轉和容錯回復，您應該熟悉 Microsoft SQL Server 2008 R2 及更新版本的複寫和容錯移轉程序。您也應熟悉 常設聊天室伺服器 服務。
