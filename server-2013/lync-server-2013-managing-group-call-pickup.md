# 在 Lync Server 2013 中管理群組來電接聽

 

_**上次修改主題的時間：** 2013-02-22_

Lync Server 2013 累計更新 (2013 年 2 月) 已導入群組來電接聽做為全新企業語音功能。群組來電接聽可讓企業語音使用者藉由撥打來電接聽群組號碼，接聽其他使用者響鈴的來電。

本節中的主題會提供您在部署中執行群組來電接聽設定工作的逐步程序。
